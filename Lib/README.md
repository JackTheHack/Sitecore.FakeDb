# Required Files

Copy the following DLLs from your Sitecore's `Website\bin`:

* HtmlAgilityPack.dll
* ITHit.WebDAV.Server.dll
* Lucene.Net.dll
* Mvp.Xml.dll
* Sitecore.Analytics.dll
* Sitecore.ContentSearch.dll
* Sitecore.ContentSearch.Linq.dll
* Sitecore.Kernel.dll
* Sitecore.Logging.dll
* sitecore.nexus.dll
* Sitecore.Buckets.dll
